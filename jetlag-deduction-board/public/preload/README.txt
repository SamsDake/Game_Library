This folder holds preloaded OSM feature data generated by:

  npm run preload

Per-country results are saved under parts/ as they download, so a re-run resumes
where it left off. The merged <feature>.json files and manifest.json are what the
app reads. The app uses any <feature>.json present even if it is not listed in
manifest.json.

Until you run the script, this folder is empty (except this file) and the app
falls back to live Overpass queries + caching.
